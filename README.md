# 公益项目 seeforme 为你所见
# 目标人群:视觉障碍群体如:盲人
# 灵感来自于刘慈欣小说: 带上她的眼睛
# 产品sologan: 当你需要时，有人替你看见。 when you need, someones see for you.
# 精神sologan: 带上你的眼睛，让世界可感。through my eyes, experience the world.

## 带上她的眼镜，重新理解这个世界。 眼耳口鼻脑

## 支持浏览器/安卓/IOS 
项目目前开发中 用户分为求助者和志愿者 求助者可以通过图片/语音/视频/文本 等形式上传求助单 
志愿者可以在大厅接单
支持文本转语音和语言转文本 文本实时朗读 更好地服务视觉障碍群体

感知可以是： 
1. 视觉 草原的辽阔/月亮的光辉 
2. 听觉 音乐/沟通
3. 味觉 家乡的味道/妈妈的饭
4. 嗅觉 香料/雨后的泥土味

5. 触觉 树的纹理/微风轻拂皮肤
6. 情感 亲情/爱情/友情
7. 思考 难题/方案/哲学

```

```
